HTML 배우기 어렵지 않아요^^

저도 HTML부터 CSS, 제이쿼리, 자바스크립트 등을 모두

독학으로 인강, 책 보며 공부했거든요

본인의 레벨에 맞는 HTML인강, 책을 선택해 꾸준히 본다면

기초 내용부터 쉽게 배워보실 수가 있을거에요

전 컴띵의 HTML인강과 생활코딩 책을 구입해 봤는데요

HTML외에 CSS, 자바스크립트, 제이쿼리 등의 웹코딩 관련

내용들까지 공부할 수 있어 좋더라구요

특히 강의는 생각보다 싸서 내용은 크게 기대안했는데

요점이랑 예제파일도 있고 선생님 설명이 꽤 괜찮았어요

초보 레벨에서 도움될만한 내용들이 있어서 님이 보기도 괜찮을 것 같아요

물론 개인차가 있으니 미리 내용 확인해보세요

그리고 공부 기간을 물어보셨는데 학원처럼 커리큘럼이 짜여져있는 것이 아니고

코딩은 범위가 넓기 때문에 얼마나 걸린다고 말씀드리기가 어려운데요

기본 내용들 공부하는 것 정도는 1~2개월 정도 기간이면 충분하시지 않을까 싶어요

HTML배우기 어렵지 않으니까 걱정마시고 인강, 책 열심히 보면서 공부하세요^^
